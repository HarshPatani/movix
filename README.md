# movix
Movies/ series info application
# Functionalities
- Infinite Scroll
- Carousels
- Search
- Loading Skeleton
- Responsive
- Filter by Genres
- Sort by Popularity/Ratings/Release date/Title
- Lazy Loading posters
- Video Player
- Watch Trailers and official clips on youtube
- Movie/Series details
- Recommended and Similar movies/series

# Tech Stack
- React
- React-router
- Redux/ redux toolkit
- Sass
- axios
- dayjs
